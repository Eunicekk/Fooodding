# 푸드트럭 상권 분석 서비스

## 주요 기능 및 플로우

### 타겟층 (B2B 서비스 st)

-   합법적으로 상업 허가를 받고, 운영 중인 푸드트럭 사장님 (= 신고 걱정 x)
-   매출액을 늘리는 것에 고민인 사장님
-   새로 사업을 시작하게 된 사장님

### 주요 기능

-   **시간대별 연령/성별별 유동인구 지도**를 통한 유동인구 많은 동네 추천
-   상업 행위 허용 가능 구역 데이터 활용
-   푸드트럭 모집공고 알림 서비스
-   푸드트럭 메뉴의 타겟층 설정 및 필터링 기능을 통한 장소 추천
-   푸드트럭 고객들의 푸드트럭 찾기 및 원격 줄서기 기능
